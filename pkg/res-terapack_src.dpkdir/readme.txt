README:

Terapack for unvanquished
Initial alpha version.

DESCRPITION:


This is the meta package for the terapack asset library.
It contains models, textures and sounds usable for unvanquished.
This package itself is only a reference for other asset packages, individual assets are inside sub packages.
Continuous updates, improvements and fixes are planned.
You can use these assets in any way you like.

Currently this contains materials for various in and outdoor settings of almost any kind.

SUBPACKAGES:

This package references tex-terapack which contains all the textures and materials, for now.
